딥러닝 모델의 주요 유형

Dec 31, 2023 in Deep Learning

딥러닝 모델 주요 유형

  1. 피드포워드 신경망 (Feed-Forward Neural Networks, FFNN)
    정의: 정보가 입력층에서 시작하여 은닉층을 거쳐 출력층으로 한 방향으로만 흐르는 신경망.
    예시: 다층 퍼셉트론(Multi-Layer Perceptron, MLP).
    특징: 순전파와 역전파를 통해 학습. 순환 또는 피드백 루프가 없음.
  2. 순환 신경망 (Recurrent Neural Networks, RNN)
    정의: 정보가 순환되는 경로를 가지는 신경망. 시퀀스 데이터(시간 순서가 있는 데이터)를 처리하기 위해 설계됨.
    예시: 기본 RNN, 장단기 메모리(LSTM), 게이트 순환 유닛(GRU).
    특징: 이전 상태를 현재 상태로 피드백하여 시계열 데이터를 모델링. 시간 종속성 및 순차적 데이터를 잘 처리함.
  3. 합성곱 신경망 (Convolutional Neural Networks, CNN)
    정의: 주로 이미지 데이터를 처리하기 위해 설계된 신경망. 합성곱 연산을 통해 특징을 추출.
    예시: LeNet, AlexNet, VGG, ResNet.
    특징: 합성곱 층과 풀링 층을 사용하여 공간적 계층 구조를 학습. 이미지 분류, 객체 검출 등에 사용됨.
  4. 트랜스포머 (Transformers)
    정의: 주로 자연어 처리(NLP)에서 사용되는 신경망. 자기 주의 메커니즘(Self-Attention Mechanism)을 사용하여 입력 시퀀스의 모든 위치 간의 관계를 모델링.
    예시: BERT, GPT, T5.
    특징: 병렬 처리가 용이하며, 긴 시퀀스 간의 관계를 잘 학습함. 최근에는 NLP 외에도 다양한 분야에서 사용됨.
  5. 자동 인코더 (Autoencoders)
    정의: 입력 데이터를 압축하여 저차원 표현(latent space)을 학습하고, 이를 다시 원래의 데이터로 복원하는 신경망.
    예시: 기본 자동 인코더, 변분 자동 인코더(VAE).
    특징: 데이터 압축, 이상 탐지, 생성 모델 등에 사용됨.
  6. 생성적 적대 신경망 (Generative Adversarial Networks, GANs)
    정의: 두 개의 신경망(생성자와 판별자)이 경쟁적으로 학습하여 데이터를 생성하는 모델.
    예시: 기본 GAN, DCGAN, StyleGAN.
    특징: 새로운 데이터를 생성하는 데 사용됨. 이미지 생성, 스타일 변환 등 다양한 생성 작업에 응용됨.
    요약
    다양성: 딥러닝 모델은 FFNN 외에도 RNN, CNN, 트랜스포머, 자동 인코더, GAN 등 다양한 구조와 특성을 가집니다.
    적용 분야: 각 모델은 특정한 문제 유형에 맞게 설계되었으며, 데이터의 특성과 문제의 요구사항에 따라 적합한 모델을 선택해야 합니다.
